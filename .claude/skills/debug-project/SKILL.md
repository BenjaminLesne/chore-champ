---
name: debug-project
description: "Investigate runtime issues by querying the database, reading server logs, and tracing code flows. Use when debugging errors, unexpected behavior, or data inconsistencies."
---

- use npm run db:studio to browse DB in GUI
- use agent-browser --help to reproduce bugs in browser (navigate, click, fill, screenshot) and retrieve console logs, network requests, and DOM snapshots
- use the next-devtools MCP tools, call nextjs_index to discover running dev servers and available tools
